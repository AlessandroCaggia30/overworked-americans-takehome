## Reproducibility

All four figures are regenerated by:

```bash
cd research
python3 build_figure_panel.py    # → figures/figure_panel.{pdf,png}    (Q1 three-panel)
python3 sim_mechanisms.py        # → figures/sim_cap.{pdf,png}         (Q4 cap simulation)
python3 sim_models_v2.py         # → figures/sim_shocks.{pdf,png},
                                 #   figures/sim_q7_eta_lambda.{pdf,png}
```

The main paper compiles with `pdflatex problem1_v2.tex` (run twice for cross-references). The companion appendix compiles independently with `pdflatex appendix_full.tex`.

Requirements: `pdflatex` with `amsmath`, `amssymb`, `booktabs`, `hyperref`, `geometry`, `caption`, `siunitx`, `titlesec`, `float`; Python 3 with `numpy`, `pandas`, `matplotlib`.
